# notepad

> An HTML5 and JavaScript partial recreation of the Windows 10 version of [Windows Notepad](https://en.wikipedia.org/wiki/Notepad_%28software%29).

## Features

- File > New file
- File > Save file
- File > Save file as
- File > Print file
- Edit > Cut
- Edit > Copy
- Edit > Select All
- Edit > Date/Time (`en_US` only)
- Help > View Help
- Word wrap/Status bar (Windows 10 1809)
